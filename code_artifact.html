Focus of Interest

왜 웹 프로그래밍(Web Programming)인가요?

웹은 세상에서 가장 접근성이 높고 유연한 플랫폼입니다. 설치 과정 없이 브라우저 주소창에 주소만 치면 개발자의 논리와 창의력이 담긴 인터페이스가 전 세계 사람들에게 그대로 전달됩니다. 코드가 실시간으로 시각적인 UI/UX로 구현되는 즉각적인 피드백 루프에 흥미를 느끼며, 이를 더 완성도 있게 구현해 내는 기술들에 깊은 관심을 두고 있습니다.

#브라우저의_가능성 #즉각적인_피드백 #인터랙티브_웹

HTML5 / CSS3 & 웹 표준

웹 문서의 뼈대를 유기적이고 구조적으로 마크업하고, 다양한 디바이스 환경에서도 무너지지 않는 유연한 반응형 화면을 스타일링하는 것에 재미를 붙이고 있습니다.

JavaScript & 동적 상호작용

정적인 웹 페이지를 살아 숨 쉬게 하는 핵심 도구로서 JavaScript에 주목하고 있습니다. 이벤트 처리, DOM 조작 등을 공부하며 사용자의 행동에 부드럽게 대답하는 인터랙션을 설계하고 싶습니다.

사용자 친화적 UI / UX

단순한 기능 구현을 넘어, 웹페이지를 방문한 사용자가 어떤 흐름에서 시각적 편안함과 조작의 유연함을 느끼는지 레이아웃 분석과 타이포그래피에 관심을 키우고 있습니다.

소프트웨어 아키텍처

대학교 전공 커리큘럼에 맞춰 객체 지향 및 프로그래밍 논리적 원리를 학습하며, 웹 개발에 접목해 확장 가능하고 깨끗한 품질의 코드를 작성하는 방법을 배우고 있습니다.
